# ✅ Todo Avançado

Aplicação de lista de tarefas (To-Do List) feita com **HTML, CSS e JavaScript**, com foco em produtividade: criar, editar, concluir, remover, além de pesquisar e filtrar tarefas. 
As tarefas ficam salvas no navegador usando **LocalStorage**, então você não perde nada ao atualizar a página.

---

## 🧩 Funcionalidades

- **Adicionar tarefa** (criação de itens)
- **Marcar como feita** (toggle de status)
- **Editar tarefa**
- **Remover tarefa**
- **Pesquisar tarefas** por texto
- **Filtrar**: todas / feitas / a fazer
- **Persistência com LocalStorage** (salva e recarrega automaticamente).

---

## 🧠 Como funciona

- A interface é construída em **HTML** e estilizada em **CSS**. 
- A lógica do app (CRUD, busca, filtro e edição) está em **JavaScript**, com eventos e manipulação do DOM.   
- As tarefas são armazenadas em `localStorage` no formato JSON, garantindo persistência entre sessões.

## ▶️ Como executar

### Opção 1 — Abrir direto no navegador
1. Baixe/clonar o projeto
2. Abra o arquivo `index.html` no navegador

### Opção 2 — Rodar com Live Server (VS Code)
1. Instale a extensão **Live Server**
2. Clique com o botão direito no `index.html` → **Open with Live Server**
